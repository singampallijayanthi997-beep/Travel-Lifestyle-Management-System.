# Travel & Lifestyle Management System

## Project Overview
The **Travel & Lifestyle Management System** is a web-based application that helps users explore and manage travel-related information efficiently. The system provides a centralized platform where users can view hotels, tourist places, tourist guides, transportation services, reviews, and manage travel budgets.

This project demonstrates the practical implementation of **Database Management System (DBMS)** concepts along with **full-stack web development** using Flask and MySQL.

---

## Features
- User Login System
- Dashboard Interface
- View Hotels and Availability
- Explore Tourist Places
- Tourist Guide Information
- Transport Services Information
- Reviews and Ratings
- Budget Planner
- DBMS Query Execution Module
- Display SQL Query Results in Table Format

## System Architecture

The project follows a **three-tier architecture**:

1. **Frontend Layer**
   - Developed using HTML, CSS, and JavaScript.
   - Provides the user interface for interacting with the system.

2. **Backend Layer**
   - Developed using Python Flask.
   - Handles application logic and API requests.

3. **Database Layer**
   - MySQL database stores all application data such as users, hotels, places, reviews, and budgets.

---

## Database Entities

The system includes the following database tables:

- Users
- Locations
- Hotels
- Places
- Tourist_Guides
- Transport
- Reviews
- Budget_Planner

These entities are connected using **Primary Keys (PK)** and **Foreign Keys (FK)**.

## Modules

### User Module
Handles user login and authentication to access the system.

### Hotel Management Module
Displays hotel details such as name, price, and availability.

### Places Module
Shows tourist attractions available in different locations.

### Tourist Guide Module
Provides information about tourist guides including contact details.

### Transport Module
Displays transportation services available for travel.

### Reviews Module
Allows users to view ratings and feedback for hotels.

### Budget Planner Module
Helps users plan and manage their travel expenses.

### Query Execution Module
Allows users to execute SQL queries such as:
- COUNT
- SUM
- MAX
- MIN
- AVG
- JOIN

Query results are displayed dynamically in table format.
